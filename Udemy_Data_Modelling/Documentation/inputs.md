| Concept    | What it is                         | Example                       |
|------------|-----------------------------------|-------------------------------|
| Entity     | Business object / "thing"         | Customer, Product, Order      |
| Attribute  | Property of an entity             | CustomerName, ProductCategory |
| Dimension  | Table of entity + its attributes  | DimCustomer, DimProduct       |
| Fact       | Event/measurement linking entities| Sales (Qty, Amount, IDs)      |

<!-- Notes:
- Entities = core business objects
- Attributes = describe entities
- Dimensions = store attributes
- Facts = store events/measures
-->
